1. UI Компоненты (Слой Представления / View)

- `CajanderPluginMainDialog` (`QDialog`) — Главное окно плагина.
    - `QTabWidget` — Контейнер вкладок.
        - **Вкладка 1: «Данные»** (`QWidget`)
            - `QComboBox` — Выбор эталонного слоя из проекта QGIS.
            - `QTreeWidget` (или `QTableWidget`) — Дерево целевых индексов с группировкой по месяцам (Март/Май/Июль) и чекбоксами.
            - `QTableWidget` — Динамическая таблица необходимых исходных каналов.
                - _Ячейки управления_: комбинированный `QComboBox` (Слои QGIS + пункт «Выбрать файл...»).
            - `QPushButton` — Кнопка запуска «Подготовить растры».
        - **Вкладка 2: «Классификация»** (`QWidget`)
            - `QHBoxLayout` — Верхняя панель управления биомами.
                - `QComboBox` — Выбор текущего активного биома.
                - `QPushButton` — Кнопка «Редактировать список биомов» (иконка карандаша).
            - `QTableWidget` — Таблица калибровочных правил для выбранного биома.
                - _Колонки_: Индекс | Направление (↑ min / ↓ max) | Коэффициент | Действия.
                - _Нижняя строка_: Интегрированная `QPushButton` с плюсиком «+ Добавить правило».
            - `QPushButton` — Кнопка запуска «Классифицировать».
        - **Вкладка 3: «Процессинг»** (`QWidget`)
            - `QLabel` — Текстовый индикатор текущей атомарной операции.
            - `QProgressBar` — Индикатор общего прогресса (0–100%).
            - `QPlainTextEdit` — Консоль логгера (Read-Only, темный фон, моноширинный шрифт).
            - `QPushButton` — Кнопка «Отмена» для прерывания фонового потока.
    - **Вспомогательные Диалоги (Модальные окна `QDialog`)**
        - `BiomeManagerDialog` — Окно редактирования списка биомов.
            - `QTableWidget` — Таблица: Название биома | `int` код цвета/класса.
            - `QPushButton` (x2) — Кнопки «Добавить биом» и «Удалить биом».
        - `RuleEditorDialog` — Окно детальной сборки/редактирования формулы.
            - `QComboBox` — Выбор индекса.
            - `QComboBox` — Выбор типа ограничения (Минимум / Максимум).
            - `QLineEdit` — Ввод числового порога (с валидатором `QDoubleValidator`).

2. Компоненты Логики и Данных (Слой Модели / Model)

- `ConfigManager` — Класс для сериализации данных в JSON (через встроенный Qt JSON или `nlohmann/json`). Отвечает за сохранение в профиль QGIS или в `.qgz` проект.
- `Structures` — Заголовочный файл со структурами данных:
    - `Rule` — Структура правила (`GeoIndex index`, `bool isMax`, `double value`).
    - `Biome` — Структура биома (`QString name`, `int code`, `std::vector<Rule> rules`).
- `RasterProcessingTask` (`QgsTask` / `QThread`) — Тяжелый вычислительный движок.
    - _Интерфейс GDAL_: Чтение/запись растров блоками (тайлами, например, 512x512).
    - _Алгоритмы_: Математические формулы индексов + алгоритм «сито» для биомов.
    - _Сигналы (Signals)_: `progressChanged(int)`, `statusTextChanged(QString)`, `logAdded(LogEntry)`.
